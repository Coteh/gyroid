# Tasks for gyroid

## build

> Builds gyroid

```sh
go build
```

## run

> Runs gyroid

### run exe

> Runs gyroid built exe

```sh
./gyroid
```

### run go

> Compile and run gyroid main.go file (doesn't build exe)

```sh
go run main.go
```

## install

> Installs gyroid to GOPATH

```sh
go install gyroid
```

## test

> Runs tests

```sh
go test ./...
```

## clean

> Cleans gyroid

```sh
rm -rf ./gyroid
```
